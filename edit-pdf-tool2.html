Frequently Asked Questions

Find answers to common questions about our PDF editor

Is it really free to edit PDFs with your tool?

Yes, our PDF editor is completely free to use. You can add text, images, shapes, annotations, and more without any cost or watermarks.

What types of edits can I make to my PDF?

You can add new text, insert images, draw shapes, highlight text, underline, strikethrough, add freehand drawings, and manage pages (add, delete, rotate). Our editor provides a comprehensive set of tools for most PDF editing needs.

Do I need to install any software to use this tool?

No installation is required. Our PDF editor works entirely in your web browser, so you can use it on any device with an internet connection.

Are my files secure when using your PDF editor?

Absolutely. We prioritize your privacy and security. All file processing happens securely in your browser, and we automatically delete your files after processing. We never store your documents on our servers.

What is the maximum file size I can edit?

We support PDF files up to 100MB. For optimal performance, we recommend files under 50MB. If you have a larger file, consider compressing it first using our PDF compressor tool.

Can I edit scanned PDFs or PDFs with images?

Yes, you can add annotations, highlights, and drawings to any PDF, including scanned documents. For editing text in scanned PDFs, we recommend using our OCR PDF tool first to convert the scanned document into searchable and editable text.
